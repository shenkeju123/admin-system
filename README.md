# Admin System

基于Spring Boot + Vue3的后台管理系统

## 项目结构

```
├── admin-system-backend    // 后端项目
│   ├── src
│   │   └── main
│   │       └── java
│   │           └── com
│   │               └── admin
│   │                   └── system
│   └── pom.xml
├── admin-system-frontend   // 前端项目
└── README.md
```

## 技术栈

### 后端
- Spring Boot 2.7.0
- Spring Security
- MyBatis-Plus 3.5.2
- JWT

### 前端
- Vue 3
- Element Plus
- Vite
- TypeScript

## 功能特性

- [x] 用户登录
- [x] JWT认证
- [ ] 用户管理
- [ ] 角色管理
- [ ] 权限管理

## 开发环境

- JDK 1.8+
- Maven 3.6+
- Node.js 16+
- MySQL 5.7+
